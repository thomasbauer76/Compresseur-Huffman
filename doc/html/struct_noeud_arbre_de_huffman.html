Detailed Description

Le type NoeudArbreDeHuffman est une structure qui contient un octet ainsi qu'une fréquence associée à cet octet. La valeur de l'octet n'a de sens que si le noeud est une feuille et cette information est stocée dans le booléen 'estUneFeuille'. Les deux derniers champs de cette structure sont 2 ArbreDeHuffman qui représentent un fils gauche et un fils droit.
